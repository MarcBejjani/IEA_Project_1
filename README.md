# IEA_Project_1
 Intelligent hand-written character recognizer agent using non-parametric learner


## Setup & Installtion

```bash
git clone https://github.com/daniazzam/IEA_Project_1/
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
